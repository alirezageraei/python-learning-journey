# Python Generators

A collection of generator examples for memory-efficient and lazy data processing.

### Features

* Infinite counter generator
* Even numbers generator
* Prime numbers generator
* Lazy list reader
* File line reader
* Error log filter
* Fibonacci sequence generator
* Generator pipeline (chaining generators together)
* Mini CSV streamer (CSV to dictionary)

### Concepts Used

* Python Basics
* yield keyword
* Lazy Evaluation
* Generator Pipelines
* File Handling
* String Parsing
