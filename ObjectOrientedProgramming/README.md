# Object-Oriented Programming Exercises

A collection of practical OOP exercises covering fundamental to intermediate concepts in Python.

## Topics Covered

- Classes and Objects
- Instance Methods and Attributes
- Inheritance (Single)
- Polymorphism
- Abstract Classes (ABC)
- Aggregation (HAS-A, independent)
- Composition (HAS-A, dependent)
- Magic Methods (`__str__`, `__len__`)

## Exercises

1. **Dog Class** – Basic class with `__init__` and instance methods
2. **Rectangle Class** – Area and perimeter calculator
3. **Bank Account** – Deposit, withdraw, balance validation
4. **Car & ElectricCar** – Inheritance with `super()`
5. **Library & Books** – Aggregation example (books exist without library)
6. **Playlist** – Magic methods `__str__` and `__len__`
7. **Animal, Dog, Cat** – Polymorphism in action
8. **Shape, Rectangle, Circle** – Abstract classes with `@abstractmethod`
9. **Car & Engine** – Composition example (engine cannot exist without car)

## Key Concepts Explained

| Concept | Meaning |
|---------|---------|
| Superclass | A class that is inherited by others |
| Common Parent | A superclass shared by multiple subclasses |
| Abstract Class | A class that cannot be instantiated |
| Aggregation | HAS-A relationship, parts can exist independently |
| Composition | HAS-A relationship, parts depend on the whole |
