# Python Decorators

A collection of practical decorator examples for learning how decorators work in Python.

### Features

* Before & After function execution
* Call counter (tracks how many times a function runs)
* Uppercase output converter
* Execution timer
* Function logger (shows name and arguments)
* Repeat function multiple times
* Type checker for arguments
* Input validator
* Multiple decorators stacking

### Concepts Used

* Python Basics
* Nested Functions
* Closures
* args and kwargs
* Function Wrapping
